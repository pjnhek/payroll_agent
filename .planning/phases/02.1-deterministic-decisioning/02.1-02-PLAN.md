Rewrite the two judgment stages that ARE the thesis into pure deterministic code (D-21-01, D-21-02, D-21-03, D-21-08): `reconcile_names` resolves exact/stored-alias matches and degrades everything else to unresolved with NO LLM call; `decide` computes a code-owned `final_action` from resolution facts + run-level collision checks + missing fields, with NO LLM call and NO confidence. Output: Pure `reconcile_names.py` + `decide.py`; deleted decide/reconcile prompts (or repurposed); a new deterministic-resolution test taxonomy in `test_reconcile.py` + `test_gate.py`.
