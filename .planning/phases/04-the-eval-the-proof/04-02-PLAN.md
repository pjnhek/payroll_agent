  CORE SCORING FUNCTION: _score_fixture(raw: dict, fixture_path: pathlib.Path) -> dict
  This function scores one fixture and returns a per-fixture result dict.

  1. Load extraction cache via _load_extraction_cache(fixture_path) → actual_extracted: Extracted
  2. Load roster via _load_roster_for_fixture(raw["from_addr"]) → roster: Roster
  3. Run the deterministic stages (PATH A — isolated, per D-07):
     submitted_names = [e.submitted_name for e in actual_extracted.employees]
     matches: list[NameMatchResult] = reconcile_names(submitted_names, roster)
     issues = validate(actual_extracted, roster, matches)
     decision = decide(actual_extracted, matches, issues)

  4. EXTRACTION SCORING (D-06 precision+recall+F1):
     actual_by_name: dict[str,ExtractedEmployee] = {_normalize(e.submitted_name): e for e in actual_extracted.employees}
     expected_emps: list[dict] = raw["expected"]["extracted"]["employees"]
     expected_by_name: dict[str,dict] = {_normalize(e["submitted_name"]): e for e in expected_emps}
     matched_names = set(actual_by_name) & set(expected_by_name)
     false_positives = set(actual_by_name) - set(expected_by_name)   # hallucinated extra employee
     false_negatives = set(expected_by_name) - set(actual_by_name)   # dropped employee
     precision = len(matched_names)/(len(matched_names)+len(false_positives)) if actual_by_name else 1.0
     recall = len(matched_names)/(len(matched_names)+len(false_negatives)) if expected_by_name else 1.0
     f1 = (2*precision*recall/(precision+recall)) if (precision+recall) > 0 else 0.0
     
     For each matched employee, score per-field (hours_regular/overtime/vacation/sick/holiday):
     A field is "correct" if both actual and expected are null, OR if Decimal(actual) == Decimal(expected).
     Never use float equality. Count field_correct/field_total across all matched employees.
     extraction_scores = {"precision": precision, "recall": recall, "f1": f1, "field_accuracy": field_correct/field_total if field_total > 0 else 1.0, "false_positives": len(false_positives), "false_negatives": len(false_negatives)}

  5. RECONCILIATION SCORING (D-03 per-NAME):
     For each entry in raw["expected"]["reconciliation"]:
       name_cat = entry["name_category"]
       submitted = entry["submitted_name"]
       expected_source = entry["expected_source"]
       expected_resolved = entry["expected_resolved"]
       Find the actual NameMatchResult with matching submitted_name (normalize both)
       Score: source_match = (actual.source == expected_source)
               resolved_match = (actual.resolved == expected_resolved)
               correct = source_match and resolved_match
       Append: {"submitted_name": submitted, "name_category": name_cat, "correct": correct, "actual_source": actual.source, "actual_resolved": actual.resolved}
     Store as reconciliation_results: list[dict]

  6. DECISION SCORING — two levels (D-10):
     exp_dec = raw["expected"]["decision"]
     action_correct = (decision.final_action == exp_dec["final_action"])
     
     gate_reasons_match: bool
       gate_reasons_contains = exp_dec.get("gate_reasons_contains", [])
       if gate_reasons_contains:
           joined = " ".join(decision.gate_reasons)
           gate_reasons_match = all(s in joined for s in gate_reasons_contains)
       else:
           gate_reasons_match = (set(decision.gate_reasons) == set(exp_dec.get("gate_reasons", [])))
     
     unresolved_match = (set(decision.unresolved_names) == set(exp_dec.get("unresolved_names", [])))
     missing_match = (set(decision.missing_fields) == set(exp_dec.get("missing_fields", [])))
     gate_struct_ok = gate_reasons_match and unresolved_match and missing_match
     
     decision_scores = {"action_correct": action_correct, "gate_struct_ok": gate_struct_ok, "final_action": decision.final_action, "expected_final_action": exp_dec["final_action"]}

  7. Return a per-fixture result dict containing:
     {"fixture_id": raw["id"], "fixture_path": str(fixture_path.name), "fixture_category": raw["fixture_category"],
      "extraction": extraction_scores, "reconciliation": reconciliation_results,
      "decision": decision_scores}

  AGGREGATION FUNCTION: _aggregate(fixture_results: list[dict]) -> dict
  Computes per-category metrics + confusion matrix.

  EXTRACTION per-category:
  Group fixture_results by fixture_category. For each category: mean F1, mean field_accuracy.
  Also compute overall extraction F1 (macro-average across fixtures).

  RECONCILIATION per-NAME-category:
  Flatten all reconciliation_results entries across all fixtures.
  Group by name_category. For each: count correct / total (the k/n fraction, D-12).
  Store as {"category": c, "correct": k, "total": n, "accuracy": k/n}.

  DECISION confusion matrix (D-11, D-12):
  Build a 2×2 matrix over all fixtures:
    actual_process_expected_process    = TP (correct process)
    actual_process_expected_clarify    = FP = false_process_count (THE HEADLINE — pays wrong person)
    actual_clarify_expected_process    = FN = false_clarify_count (annoying, not dangerous)
    actual_clarify_expected_clarify    = TN (correct clarify)
  false_process_rate = false_process_count / (false_process_count + TP) if (false_process_count + TP) > 0 else 0.0
  
  Per-fixture-category decision: for each category, report as "k/n correct" (D-12 — use fractions, not percentages at small n).

  Return aggregated = {
    "extraction_overall_f1": float,
    "per_category_extraction": {cat: {f1, field_accuracy}, ...},
    "per_category_reconciliation": [{category, correct, total, accuracy}, ...],
    "confusion_matrix": {"true_process": int, "false_process": int, "false_clarify": int, "true_clarify": int, "false_process_rate": float},
    "per_category_decision": {cat: {"correct": k, "total": n, "fraction": "k/n"}, ...},
    "rigor_gate_struct_accuracy": float (fraction of fixtures where gate_struct_ok=True)
  }
