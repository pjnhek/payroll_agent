  CORRECTNESS FLOOR (D-16 critical rule):
  After getting a raw score from the LLM, apply this deterministic check:
  1. Load all roster employee full_names via seed(dry_run=True).
  2. Check if any roster full_name (case-insensitive) appears in the draft text.
  3. If a roster name appears AND it doesn't match the fixture's expected_employee_name, apply floor: final_score = min(raw_score, 1).
  4. Log the floor application with a clear message.

  FUNCTION: judge_draft(draft_text: str, fixture_id: str, expected_employee_full_name: str | None) -> dict
  - Checks allow_live_llm — raises SystemExit if False
  - Calls call_text(tier="draft", messages=[system_rubric_msg + user_msg_with_draft_text], temperature=0.3)
  - Parses the integer score from response (1-5), defaults to 0 on parse failure
  - Applies correctness floor (D-16)
  - Returns: {"fixture_id": fixture_id, "raw_score": raw, "final_score": final, "floor_applied": bool, "notes": str}

  STANDALONE USAGE:
  if __name__ == "__main__":
    The script expects draft text files to exist at eval/drafts/{fixture_id}_draft.txt
