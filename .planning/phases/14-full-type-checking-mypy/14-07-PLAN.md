<objective>
Bring the second third (18 files including `conftest.py`, ~8945 lines, balanced group 2) of `tests/` to zero mypy errors under Plan 14-01's `tests.*` relaxed override.

Purpose: `conftest.py` (1226 lines) is the shared fixture file every other test file in the suite depends on at RUNTIME via pytest's fixture injection — but NOT at mypy-analysis time: mypy only follows imports, and test modules do not import conftest.py, so annotating a fixture's return type here does not automatically type the unannotated `def test_x(fake_fixture):` parameters in other files (those parameters remain `Any` unless explicitly annotated). What this plan DOES deliver: conftest.py's fixture implementations themselves become fully type-checked (wrong internal shapes, bad return values, and mismatched fake-object attributes get caught), and any test file that chooses to annotate a fixture parameter gets a concrete type to reference. This corrected framing addresses the Codex review — the old plan overstated cross-file fixture inference.

Output: Zero mypy errors across this plan's 18 test files; full hermetic suite green with zero assertion changes.
</objective>
