<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Public internet → /webhook/inbound | Unauthenticated POST; must reject unsigned payloads before ANY pipeline work |
| gateway.py → resend.EmailsReceiving.get | Authenticated outbound API call (resend_api_key); body is not sensitive but the call leaks existence of inbound email |
| gateway.py → resend.Emails.send | Sends real money-themed emails; D-13c reserved row ensures record even if send fails |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-04-01 | Spoofing | /webhook/inbound — forged POST without svix-signature | mitigate | gateway.verify() is step 0a — ValueError → HTTP 400 before ANY extraction, parse, or pipeline work (D-17 HIGH) |
| T-06-04-02 | Tampering | Duplicate inbound delivery re-runs decide.py (false-process) | mitigate | Dedup on RFC message_id via email_messages.uq_message_id UNIQUE + ON CONFLICT DO NOTHING; runs AFTER parse_inbound fetch (to have the real RFC ID) but BEFORE pipeline (D-13 HIGH) |
| T-06-04-03 | Repudiation | send_state='reserved' before resend.Emails.send crash | mitigate | D-13c intent-before-side-effect: DB row written first; on crash the row is reserved (not lost); retry can detect and complete (D-14 Integrity) |
| T-06-04-04 | Tampering | Threading corruption from dropped/duplicated delivery | mitigate | send_outbound reads and appends the persisted References chain from DB rows (not ephemeral webhook state) — durable threading (D-14) |
| T-06-04-05 | Spoofing | Replay attack (old webhook replayed within TTL) | mitigate | resend.Webhooks.verify() enforces 5-minute timestamp tolerance via DEFAULT_WEBHOOK_TOLERANCE_SECONDS (SDK default, verified from source) |
| T-06-04-06 | Tampering | Raw provider headers dict passed downstream (provider blob) | mitigate | parse_inbound returns canonical InboundEmail only; headers_lower extraction + typed fields; no dict downstream (D-18) |
| T-06-04-SC | Tampering | resend SDK calls (EmailsReceiving.get, Emails.send) | accept | resend 2.32.2 legitimacy verified (RESEARCH.md Package Legitimacy Audit); pinned exact version in pyproject.toml |
</threat_model>
