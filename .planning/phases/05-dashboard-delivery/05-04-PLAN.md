## Phase Goal

**As an** operator, **I want to** approve payroll runs through a race-safe gate, **so that** duplicate approvals are structurally impossible and a failed delivery never silently strands a run.

<objective>

After this plan: generate_paystub_pdf can produce a valid PDF from a PaystubLineItem, and compose_confirmation degrades to a deterministic floor on any LLM failure — so the approval can never strand due to a draft failure or cold-dyno LLM latency.

Purpose: These are the pure-function building blocks for the Beat 1 delivery path (Plan 05).
Output: app/pipeline/pdf.py (new), compose_confirmation + confirmation_subject in compose_email.py (added).
</objective>
